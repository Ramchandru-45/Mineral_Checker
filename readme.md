# Mineral Checker Application

## Overview

The Mineral Checker Application helps users analyze and monitor the mineral levels in their body. It provides an easy-to-use interface for inputting test results and offers recommendations based on the analysis. It uses saliva as a medium for testing the levels of mineral in body.

## Features

- Input mineral test data (e.g., calcium, iron, magnesium)
- View detailed mineral level reports
- Export reports for medical consultations

## Installation

1. Download and Install JDK Environment on your respective platform.
2. Run jar file using cmd with command 
    `java -jar MineralChecker.jar`
3. Turn on the device by charging it.

## Usage

1. Turn the device on.
2. Connect your Desktop or Laptop to the WiFi Access point provided by device.
3. Launch the application.
4. Enter your Personal info in the provided fields.
5. Can also open your profile if it is saved beforehand.
6. Click on 'Add Sample' to add raw saliva data.
7. Below there click 'test' aside the selected mineral.
8. Wait for the device to check the values and application to calculate the result.
9. Click on Generate Result to view the final overview of the test result.

## Requirements

- JDK Environment (version 11 or above)
- Windows (10 or above), Linux or Mac (only jar file can run for every platform)
- ESP32 is Mandatory (Code will be in "Source code/device code")
